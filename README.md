# ShareKakeibo/共有家計簿

日々の収支を共有できる家計簿アプリ

# DEMO

ユーザが追加した収支金額が、円グラフでひと目でわかります。

<img width="367" alt="CHART" src="https://user-images.githubusercontent.com/96103285/205615480-607db3aa-c88a-4f40-9eb0-556c3218afc7.png">

# Features

ShareKakeibo/共有家計簿のセールスポイント

・グループ機能
・カレンダー表示機能
・グラフ表示機能
・メモ機能
